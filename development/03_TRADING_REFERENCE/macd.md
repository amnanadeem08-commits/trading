# MACD

**Type:** Documentation only — not imported by production code.
**Product:** Khaldun Trade / TIOS

## Summary

Moving Average Convergence Divergence tracks relationship between two EMAs.

## Key Points

- Signal-line crossovers and histogram shifts are common triggers
- Lagging by design
- Works best with trend context
