# Project Vision — Khaldun Trade

## What This Is

**Khaldun Trade** is an AI-powered trading intelligence platform.

**TIOS** (Trading Intelligence Operating System) is the permanent engineering, architecture, documentation, roadmap, testing, and project management system that controls development of that platform.

TIOS is the single source of truth for every future feature, sprint, validation, release, documentation update, and architectural decision.

Every future coding task must begin by reading TIOS.

## What This Is Not

- Not a trading bot
- Not “another dashboard”
- Not financial advice
- Not a system that claims guaranteed profits

## Current Objective

**Signal generation and paper trading.**

Future versions may include broker integrations and automated execution. Those capabilities must remain **disabled** until explicitly approved in TIOS release documents.

## Product Principles

1. Intelligence and validation first; automation last.
2. Paper trading before any live broker path.
3. Architecture and tests are non-negotiable.
4. Documentation stays synchronized with the codebase.
5. No profit guarantees. No advice framing. Cautious language in all user-facing output.

## Safety Disclaimer

This platform provides analysis and decision-support tooling only. Models and LLMs can be wrong with high confidence. Humans remain responsible for all capital decisions.
